## ğŸš€ Why It Matters

### The Gig Economy is the Future

- **70M+** gig workers in America (and growing)
- **40%** of the U.S. workforce has gig income
- **$1.3 trillion** contributed to the economy annually
- **36%** of Gen Z workers are freelancing

Traditional employment is becoming the alternative, not the default. Yet every major financial institution still treats gig work as a side hustle.

### The Financial Gap is Real

Gig workers face unique challenges that traditional finance ignores:

- **Variable income** makes budgeting and credit applications difficult
- **Quarterly estimated taxes** catch most people off guard
- **Missed deductions** cost thousands per year
- **No employer benefits** means finding your own health insurance and retirement
- **Multiple income streams** make tracking earnings nearly impossible

**Portable solves all of these problems in one platform.**

### The Market is Wide Open

Competitors fall into three categories:

1. **Traditional accounting software** (QuickBooks, FreshBooks) - Built for businesses with clients and invoices, not platform workers
2. **Consumer budgeting apps** (Mint, YNAB) - Don't understand business expenses or tax obligations
3. **Tax-only software** (TurboTax, H&R Block) - Only useful once a year, no real-time planning

**No one has built a financial platform specifically for gig workers. We're the first.**

## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+
- Supabase account (free tier works perfectly)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/portable.git
   cd portable
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Setup environment variables**:
   ```bash
   cp .env.example .env.local
   # Add your Supabase URL and anon key
   ```

4. **Run database migrations**:
   ```bash
   # Execute the SQL in database/supabase-migration-portable.sql in your Supabase SQL editor
   ```

5. **Start development server**:
   ```bash
   npm run dev
   ```

6. **Access the app**:
   - Visit: `http://localhost:3000`
